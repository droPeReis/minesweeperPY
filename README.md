# minesweeperPY
A simple minesweeper game i created on python while learning pointers and arrays manipulation. Enjoy. 


## Instructions 

install a recent version of Python and the wxpython library, then run main.py
You can change the themeby setting the  THEME variable in main.py by one the Themes file.
